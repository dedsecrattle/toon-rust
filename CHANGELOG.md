# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [0.1.1] - 2024-11-08

### Added
- GitHub Actions CI/CD workflows
- Documentation website deployment
- Contributing guidelines
- Code of Conduct
- Security policy
- Issue and PR templates

### Changed
- Improved README with badges and better structure

## [0.1.0] - 2024-11-08

### Added
- Initial release
- Full TOON specification v1.4 support
- Standalone API with `serde_json::Value`
- Serde-compatible API
- Customizable delimiters (comma, tab, pipe)
- Length markers and indentation options
- Strict validation mode
- Comprehensive test suite
